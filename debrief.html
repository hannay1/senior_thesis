The actual goals of the project are three fold:

  • To assess whether a rogue access point was capable of being run on the CC network. Looks like that isn’t a problem so far.
  • To assess the security of the passwords used by people in their various on line accounts. This was done automatically via a computer program that decrypted all the secure traffic sent from your machine to the rest of the network, using the certificate file you installed earlier, reading (but not recording) the decrypted passwords, and doing some analysis on them (before throwing the passwords themselves away).
    • DO NOT WORRY: YOUR PASSWORDS ARE STILL SAFE. NO ONE HAS RECORDED THEM, nor do I know your actual login credentials due to the anonymous ID being used & the fact that no usernames/emails were recorded. However, the following details have been recorded anonymously using the ID given to you at the start of this experiment:
      • The “strength” of your passwords:
        • Through various calculations, an indication as to how difficult it would be to attempt to brute force (relentlessly guess) your passwords was found. Longer passwords with more random characters are obviously a lot more difficult to guess via brute force.
      • The number of shared passwords used in your various accounts. People sometimes tend to use one password for multiple accounts. This is obviously a bad idea.
  • To assess the level of security offered by the browser you used to browse the Internet while connected to this access point.
    • While you were connected to the network, a piece of code was injected into every page you visited. This code is used by a program that records some meta-data on your browser.
    • NO PERSONAL DATA WAS TAKEN BY THIS PROGRAM, AND NOTHING HARMFUL WAS CARRIED OUT ON YOUR BROWSER
    • The data collected by this program included things like the version number of your browser, your operating system, and other boring stuff.

    • These pop-ups you may or may not have seen did not collect any personal data from you, nor did they alter you system/browser in any way.
